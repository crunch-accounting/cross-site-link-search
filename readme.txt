Connects the link search to other wordpress installations to facilitate linking between sites you control.

== Description ==

This plugin allows multiple wordpress sites to share each other's link search functionality so that post
authors can easily link between blogs.

The plugin must be installed on all of the sites which are sharing links. For security they must all be
configured with the same key.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Unzip `cross-site-link-search.zip` in the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin in `Settings` > `Cross Site Link Search`
